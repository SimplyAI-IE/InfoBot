# 📊 Pension Guru App

Fully deterministic forecasting tool for retirement planning.

---

## 🎯 Features

- No GPT in computation path
- All logic is rule-based
- Uses `conversation_flow.yaml` to simulate sessions

---

## 🔍 Inputs Required

- Age
- Income
- Contribution Rate
- Retirement Age
